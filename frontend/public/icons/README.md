# PWA Icons

This directory will contain the Progressive Web App icon files in various sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You'll need to generate these icons from your logo and add them here before deploying the application.
